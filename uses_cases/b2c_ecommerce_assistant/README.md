# B2C Ecommerce Assistant (Runnable Demo)

## Start server

```bash
python use_cases/b2c_ecommerce_assistant/server.py
```

Server endpoint:
- `http://127.0.0.1:8103/mcp`

## Run smoke test

```bash
python use_cases/b2c_ecommerce_assistant/smoke_test.py
```

What it tests:
1. create order
2. get order status
3. recommend upsell
4. request return
