# B2B Support Copilot (Runnable Demo)

## Start server

```bash
python use_cases/b2b_support_copilot/server.py
```

Server endpoint:
- `http://127.0.0.1:8101/mcp`

## Run smoke test

```bash
python use_cases/b2b_support_copilot/smoke_test.py
```

What it tests:
1. create ticket
2. classify ticket
3. suggest resolution
4. close ticket
